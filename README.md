# AdultGetnAI Landing Page

Vite, React, TypeScript, Tailwind CSS landing page prepared for Vercel deployment.

## Local Development

```sh
npm ci
npm run dev
```

## Production Build

```sh
npm run build
```

The production output is generated in `dist`.

## Vercel

Vercel can deploy this repository directly from `main`.

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
